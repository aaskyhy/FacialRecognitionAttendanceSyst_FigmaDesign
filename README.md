# FacialRecognitionAttendanceSyst_FigmaDesign
Figma UI/UX Design: 
https://www.figma.com/design/9yHEC5SQFfU0hqiQ0rnEqy/Face-Recognition-Student-Attendance-System?t=qw7FoNzDKN6VMpxn-1

The Facial Recognition Student Attendance System is designed to automate the traditional attendance process using facial recognition technology.
The system reduces manual attendance marking errors and improves efficiency by allowing lecturers to track student attendance digitally through a web-based platform and a Python-based recognition system.

It consists of 2 main components:
1. Python Desktop System (Command Prompt)
Student registration
Facial recognition for attendance marking
Login and logout functionality
Attendance recording and storage

2. Web-Based System (PHP)
Lecturer dashboard
View attendance records
Monitor student attendance percentage
Login and logout system

System Architecture
Frontend (Web): Lecturer interface for viewing attendance
Backend (Python): Facial recognition & attendance processing
